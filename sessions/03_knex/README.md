# TODOs backend

### Installation

Install command line tools globally:
```
npm install -g knex nodemon
```

Install local dependencies
```
npm install --save express body-parser express-prettify sqlite3
```

### Migrate Database

Update database scheme:
```
knex migrate:latest
```

### Run server
```
nodemon server.js
```
